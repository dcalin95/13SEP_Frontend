📸 Chrome Screenshot Master Guide

🎯 Demo Area pentru Screenshot Testing

⚡ METODA 1: Windows Snipping Tool

Ctrl + Shift + S

  1. Apasă Ctrl + Shift + S
  2. Ecranul devine gri
  3. Desenează zona pe care vrei s-o capturezi
  4. Screenshot-ul se deschide în Snipping Tool
  5. Salvează cu Ctrl + S
💡 Perfect pentru: Screenshots rapide, zone specifice

🔧 METODA 2: Chrome DevTools (PROFESIONAL)

F12 → Ctrl + Shift + P → "screenshot"

  1. Deschide DevTools cu F12
  2. Apasă Ctrl + Shift + P (Command Palette)
  3. Scrie "screenshot"
  4. Alege din opțiuni:
    • 📱 Capture area screenshot - Selectezi zona
    • 🖥️ Capture full size screenshot - Întreaga pagină
    • 👁️ Capture screenshot - Zona vizibilă
    • 📱 Capture node screenshot - Element specific
💡 Perfect pentru: Screenshots profesionale, întreaga pagină, elemente specifice

🎯 METODA 3: Element Specific

F12 → Right-click element → "Capture node screenshot"

  1. Deschide DevTools cu F12
  2. În tab-ul Elements
  3. Click dreapta pe elementul HTML
  4. Selectează "Capture node screenshot"
  5. Se descarcă automat PNG-ul
💡 Perfect pentru: Componente React specifice, debugging UI

📱 METODA 4: Mobile/Responsive Screenshots

F12 → Ctrl + Shift + M → Screenshot

  1. Deschide DevTools cu F12
  2. Toggle Device Toolbar cu Ctrl + Shift + M
  3. Selectează device-ul (iPhone, iPad, etc.)
  4. Folosește Ctrl + Shift + P → "screenshot"
  5. Capture cu dimensiuni mobile
💡 Perfect pentru: Testare responsive, demo mobile

🚀 METODA 5: Full Page Screenshot (AWESOME!)

Capturează întreaga pagină, inclusiv partea care nu e vizibilă!

  1. Apasă F12
  2. Apasă Ctrl + Shift + P
  3. Scrie "full" și selectează "Capture full size screenshot"
  4. Chrome va face scroll automat și captura totul!
  5. Se descarcă PNG de înaltă calitate
💡 PERFECT pentru BitSwapDEX: Capturează toată pagina presale, inclusiv footer-ul!
