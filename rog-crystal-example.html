🔤 ROG Crystal Font Implementation

This example demonstrates the complete implementation of the ROG Crystal font with various effects and styling options. The font provides a futuristic, gaming-oriented aesthetic perfect for modern web interfaces. 
/* ROG Crystal Font Implementation */
@font-face {
    font-family: 'ROG Crystal';
    src: url('./fonts/rog-crystal-regular.woff2') format('woff2'),
         url('./fonts/rog-crystal-regular.woff') format('woff');
    font-weight: normal;
    font-style: normal;
    font-display: swap;
}

.rog-text {
    font-family: 'ROG Crystal', 'Orbitron', 'Arial Black', sans-serif;
    letter-spacing: 1px;
    text-rendering: optimizeLegibility;
}

📋 Implementation Notes

Font Loading: Uses font-display: swap for optimal performance
Fallbacks: Orbitron → Arial Black → sans-serif
Effects: Gradient text, neon glow, crystal cut animations
Responsive: Adaptive sizing for all screen sizes
Accessible: Respects user preferences for motion and contrast
